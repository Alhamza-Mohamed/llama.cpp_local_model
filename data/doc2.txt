Backpropagation and Gradient Descent

Training a neural network requires adjusting its weights so that its predictions get closer to the true labels. This is done through a process called backpropagation combined with an optimization algorithm called gradient descent.

Loss Functions
A loss function measures how wrong the model's predictions are. For regression tasks, Mean Squared Error (MSE) is commonly used. For binary classification, Binary Cross-Entropy is the standard choice. For multi-class classification, Categorical Cross-Entropy is applied. The goal of training is to minimize the value of the loss function.

Gradient Descent
Gradient descent works by computing the gradient of the loss with respect to each weight and then updating the weight in the opposite direction of the gradient. The learning rate controls how large each update step is. A learning rate that is too high causes the optimization to overshoot and diverge. A learning rate that is too low causes very slow convergence.

Variants of gradient descent include Batch Gradient Descent, which computes gradients over the entire dataset at once. Stochastic Gradient Descent (SGD) updates weights after each individual sample. Mini-Batch Gradient Descent is the most common approach and updates weights after processing a small batch of samples.

Backpropagation
Backpropagation is the algorithm used to compute gradients efficiently. It applies the chain rule of calculus to propagate the error signal from the output layer back through the network to each weight. For each layer, it computes how much that layer contributed to the final loss, and uses this to calculate the weight gradient.

The steps are: perform a forward pass to get predictions, compute the loss, then propagate the error backwards layer by layer to compute all gradients, and finally update each weight using the gradients.

Advanced optimizers like Adam, RMSProp, and Adagrad build on gradient descent by maintaining adaptive learning rates for each parameter, which leads to faster and more stable convergence in practice.
