Neural Networks: Fundamentals and Architecture

A neural network is a computational model inspired by the structure of the human brain. It consists of layers of interconnected nodes called neurons. Each neuron receives input, applies a weighted sum, adds a bias, and passes the result through an activation function.

The three main types of layers are the input layer, hidden layers, and the output layer. The input layer receives raw data such as pixel values or numerical features. Hidden layers perform intermediate transformations, and the output layer produces the final prediction.

Weights and Biases
Each connection between neurons has an associated weight that controls how much influence one neuron has on another. Biases allow the network to shift the activation function, giving the model more flexibility. During training, both weights and biases are adjusted to minimize the loss function.

Activation Functions
Activation functions introduce non-linearity into the network, allowing it to learn complex patterns. Common activation functions include ReLU (Rectified Linear Unit), which returns zero for negative inputs and the input itself for positive values. Sigmoid maps inputs to values between 0 and 1, making it useful for binary classification. Tanh is similar to sigmoid but outputs values between -1 and 1. Softmax is used in the output layer for multi-class classification, converting raw scores into probabilities that sum to one.

Forward Pass
During a forward pass, data flows from the input layer through each hidden layer to the output layer. At each layer, the input is multiplied by the weight matrix, the bias is added, and an activation function is applied. The result is passed to the next layer until the output is produced.

The depth of a network (number of hidden layers) and the width (number of neurons per layer) are key hyperparameters. Deeper networks can represent more complex functions but require more data and computational resources to train effectively.
